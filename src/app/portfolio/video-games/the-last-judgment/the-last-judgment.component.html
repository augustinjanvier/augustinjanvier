Jouer

Description

Bienvenue dans The Last Judgment (en français : Le dernier jugement).

The Last Judgment est un petit jeu de cartes dans lequel vous êtes responsable du purgatoire : c'est à vous que revient la lourde tâche d'orienter les âmes des défunts vers l'enfer ou le paradis.

Pour ce faire, effectuez des combinaisons de 3 cartes, qui détermineront la sentance du pauvre bonhomme qui vous fait face. Découvrez les différentes combinaisons de carte, vous pouvez vous aider du tutoriel en jeu afin d'avoir quelques bases.

L'objectif est d'obtenir le plus haut score en jugeant 20 âmes. Mais attention, choisissez votre camp : les bons ou les méchants. Car plus vous êtes bons, plus votre score augmente, mais diminuera si vous êtes mauvais, et inversement !

Réalisation

Ce jeu a été réalisé lors de la Mini Jam 69 fin 2020.

Nous avions 72 heures pour réaliser un jeu sur le thème "Hell", avec la contrainte "Cards must be a mechanic".

J'ai réalisé ce jeu avec le moteur de jeu Unity 3D (code en C#). J'ai dessiné les graphismes en pixel art avec Aseprite. La musique et les effets sonnores sont aussi de ma création, avec Ableton.

Créer ce jeu était un véritable challenge car je n'avais que 3 jours pour implémenter la mécanique du jeu de cartes, les graphismes et le son. Le résultat est plutôt réussi selon moi, malgré les règles du jeu un peu floues d'après les retours.

C'est également la première fois que quelqu'un réalise une vidéo de gameplay sur une de mes créations ! Un participant de la game jam avait choisi de jouer, filmer et commenter les différents jeux de l'événement, il y a donc une vidéo dédiée à The Last Judgment que vous pouvez retrouver ci dessous.
