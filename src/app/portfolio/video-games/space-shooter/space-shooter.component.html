Space Shooter

Jouer

Description

Bienvenue dans Space Shooter.

Essayez de survivre aux vagues d'aliens qui foncent droit sur vous, et ne vous faites pas tirer dessus. A bord de votre vaisseau, réalisez le meilleur score !

Eliminez les yeux extraterrestres afin de vous frayer un passage et survivre le plus longtemps possible.

Utilisez les touches AWSD afin de déplacer le vaisseau, orientez le vaisseau et tirez avec le clic de la souris !

Note pour les utilisateurs de clavier AZERTY : Pour pouvoir utiliser des touches plus classiques (ZQSD), vous pouvez changer la langue de votre clavier, le plus souvent avec le raccourci Alt + Shift.

Réalisation

Ce jeu a été réalisé lors de la Untitled Game Jam #38 en 2020.

Nous avions 10 jours pour créer un jeu sur le thème "Minimalism", 23 personnes ont participé. Le nombre de participants n'ayant pas été assez conséquent, les jeux n'ont pas été notés. Malgré les 10 jours disponibles, de mon côté je n'ai pu travailler seulement 3 jours sur ce jeu, ce qui m'a tout de même permis de réaliser un jeu d'arcade fonctionnel.

J'ai réalisé ce jeu avec le moteur de jeu Unity 3D (code en C#). J'ai dessiné les graphismes moi même avec le logiciel de dessin vectoriel Affinity Designer, afin de tester ce style graphique minimaliste et épuré pour être dans le thème.

La musique a été réalisée par mes soins avec le logiciel Ableton, et les effets sonnores sont directement issus de ma bouche ! J'avais reçu un nouveau micro pour mon anniversaire...

Ce jeu m'a permis d'expérimenter avec les effets de lumière dans un environnement 2D.
